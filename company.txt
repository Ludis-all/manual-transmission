[b]Applying Corporation Codes[/b]
Under international law corporations must apply their corresponding identification code to security codes.
- For [b]WEST[/b] and [b]CENTER[/b] nations, the [b]Last Digit[/b] from the code is used for identification
- For [b]EAST[/b] nations, the [b]2nd Digit[/b] from the code is used for identification
